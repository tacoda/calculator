## Usage

```sh
$ calculator help
calculator 0.1.0
A simple command line calculator

USAGE:
    calculator <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    abs        Find the absolute value of a number
    add        Add a list of numbers
    cbrt       Find the cube root of a number
    cos        Find the cosine of a number
    cube       Find the cube of a number
    div        Divide two numbers
    exp        Find the value of the exp function of a number
    fact       Find the factorial of a number
    help       Prints this message or the help of the given subcommand(s)
    ln         Find the natural logarithm of a number
    log        Find the base 10 logarithm of a number
    mean       Find the mean of a list of numbers
    median     Find the median of a list of numbers
    mode       Find the mode of a list of numbers
    mult       Multiply a list of numbers
    percent    Compute a percentage of a value
    pow        Find the power of a base and exponent
    root       Find the nth root of a number
    sin        Find the sine of a number
    sqrt       Find the square root of a number
    square     Find the square of a number
    sub        Subtract two numbers
    tan        Find the tangent of a number

```
